## 1. Explain the difference between:

### i. `<strong>` tag and `<b>` tag

- `<strong>` tag is used to emphasize and bolden important text.
- `<b>` tag only styles text as bold.

### ii. `<em>` tag and `<i>` tag

- `<em>` tag stands for emphasis, and is used to add emphasis to a text by italizing it. Just like in a book
- `<i>` tag stands for italics, and is used to style text as italics.

## 2. What is Semantic HTML, and List the importance.

Semantic HTML means using HTML tags that clearly describes the meaning and purpose the content inside. Semantic HTML is direct, and at a glance anyone reading the code can tell what it does.

### Importance of Semantic HTML

i. Easier maintenance
ii. Improves accessibility
iii. Cleaner and more readable code
