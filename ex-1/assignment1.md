1. Explain the difference between:
i. <strong> tag and <b> tag.

Answer
Strong tag is used to define text with strong sematic importance. The content aids both screen readers and search engines. While,  the bold tag is purely just for styling the web page it has no significance to the search engine 

ii. <em> tag and <i> tag.
Answer
Both tags display text in italics. Em tag is used according to where the emphasis is placed in the sentence, it adds semantic meaning. While i tag is used for only styling without semantic meaning.
